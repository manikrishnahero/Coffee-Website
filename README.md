## ☕ Coffee Website

A responsive **Coffee Website** built with pure **HTML and CSS**,
featuring a modern, visually appealing interface with smooth animations
and a fully responsive layout across all devices.

### 🚀 Features
- ☕ Modern & clean coffee-themed UI design
- 🎞️ CSS animations & smooth transition effects
- 📱 Fully responsive — desktop, tablet & mobile
- ♿ Accessibility & semantic HTML best practices
- 🎨 Consistent typography, color palette & spacing
- 🖥️ Cross-browser compatible layout

### 🛠️ Tech Stack
| Technology    | Usage                              |
|---------------|------------------------------------|
| HTML5         | Semantic structure & markup        |
| CSS3          | Styling, layout & animations       |
| CSS Flexbox   | Responsive horizontal layouts      |
| CSS Grid      | Multi-column responsive sections   |
| CSS Animation | Transitions & keyframe effects     |
| Media Queries | Mobile & tablet responsive design  |

### 📦 Installation

\`\`\`bash
git clone https://github.com/manikrishnahero/coffee-website.git
cd coffee-website
\`\`\`

Then open **index.html** directly in your browser — no build step needed.
